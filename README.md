Structure:
  1. Importing Data
  2. Visualizing Data
  3. Selecting zone of interest
  4. K-means Clustering
  5. PCA Decomposition
  6. K-means Visualization of the PCA decomposition
  7. Creation of an animated GIF for presentations.
